# safran-time-tracker-web

Frontend de SAFRAN TIME TRACKER (React + TypeScript + Vite). Voir la documentation du projet à la racine du dépôt : `README.md`, `CLAUDE.md`, `docs/ARCHITECTURE.md`, `docs/CONVENTIONS.md`.

## Commandes

- `npm run dev` — serveur de développement (proxy `/api` vers l'API locale, voir `vite.config.ts`)
- `npm run build` — build de production (`tsc -b && vite build`)
- `npm run lint` — ESLint
- `npm run format` / `npm run format:check` — Prettier
- `npm run preview` — prévisualisation du build de production
